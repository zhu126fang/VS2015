CPX-CEC Webserver

CPX-CEC variable server

The CPX-CEC variable server displays the values of CoDeSys program variables using a web interface.

  • Usage of the web interface
  • Setup in CoDeSys

Usage of the web interface

The interface of the variable server should by default look like in the screenshot below:

Explanation:

  1. Status: Displays the state of the variable server. (See state messages explained below)
  2. Interval: Interval in which the variables are polled from the server.
  3. Start: Start polling variables and set server to active.
  4. Stop: Stop polling variables and set server to inactive.
  5. Edit: Configure the list of variables that is being polled from the server.
  6. Table: List of variables that is configured to be polled.

Status information

The variable server displays its state in the field at 1. There are three states:

  • Not running: The server is not set up on the CoDeSys side or the PLC is in STOP mode. (See: Setup in CoDeSys)
  • Inactive: The server is set up correctly but currently no values are being polled.
  • Active: The server is set up correctly and values are being polled from it.

If the table just shows No variables are configured at 6, click the Edit button to configure a list of variables to monitor.

Configuring the list of variables

Upon entering the edit page the symbol file of the PLC is downloaded and parsed. Depending on the size of your symbol file this can take a while. If no symbol file is found or the symbol file is empty an error message will be displayed.

Explanation:

  1. Search: Applies a filter to the variables listed in the table below.
  2. Mark all/none: Checks the check boxes for each variable currently listed in the table.
  3. Table: List of variables in the symbol file.
  4. Check: Displays how many variables have been marked.
  5. Apply: Write the selection of variables to the PLC as a configuration file.
  6. Cancel: Return to the regular web server page without writing a configuration file.
