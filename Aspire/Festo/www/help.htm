CPX-CEC Webserver

Customizing the CPX-CEC web server

The CPX-CEC web server can easily be modified to your requirements. An example might be to keep some documentation about a machine directly on the device. The recommended way to do this:

  1. Log into the device via telnet
  2. Enable the ftp server
  3. Download the files to modify
  4. Upload user-specific documents
  5. Modify downloaded files
  6. Re-upload modified files

Telnet & Ftp

To log onto the device open a DOS shell (Windows + R, open cmd.exe). Then type:

telnet DEVICEIP

The login is root and the default password is Festo. Once logged in, simply type startftp. Then log out with exit. Now the PLC's file system is accessible with any FTP client. The user and password are the same as for the telnet log in. When uploading documents make sure to put them somewhere in the /ffx/www/ to make them accessible for the web server.

What files to modify

The suggested way to link your own files in the existing web interface is in the sidebar of the main page. This is located in /ffx/www/sidebar.php. In order to link your own documents in the sidebar insert a section like the one sampled below:

<h3>MySection</h3> <div class="sidesection"> <ul> <li><a href="path/foo.htm">Documentation</a></li> <li><a href="path/bar.htm">Part list</a></li> </ul> </div> <!-- sidesection -->
