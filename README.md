1. 开始页中打开</p>

2. 文件菜单中打开</p>
	文件菜单-->打开-->在源代码管理器中打开</p>

###项目提交
1. 提交到本地</p>
	视图-->团队资源管理器-->更改-->添加提交信息-->提交到本地</p>
2. 推送到远程</p>
	视图-->团队资源管理器-->同步-->推送</p>
